Constants definition for the pre-defined choices and report elements and items.

Package Specification

This package contains the all the pre-defined choice constants and internal representation name of all the report elements and items.

The key classes in this package include:

  • {@link org.eclipse.birt.report.model.api.elements.DesignChoiceConstants} is the interface class to define all the choice constants.
  • {@link org.eclipse.birt.report.model.api.elements.ReportDesignConstants} provides generic access to a user-defined property. The class provides some getter/setters to the name, type, choices, and so on.
