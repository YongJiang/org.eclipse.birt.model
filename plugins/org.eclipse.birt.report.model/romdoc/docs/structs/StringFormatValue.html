StringFormatValue

Used to format a string value.

Description

Provides a set of pre-defined string formatter, or user can define a customized formatter given a pattern.

category

Provides a list of pre-defined string formatter, upper, lower, zipCode, etc.

Description

Allow user to choose from a list of pre-defined string formatter. Use a certain formatter to convert a "string" typed value using the given formatter.

pattern

Pattern of the formatter.

Description

Pattern of the string time formatter.
