DateTimeFormatValue

Used to format a datetime value.

Description

Provides a set of pre-defined date time formatter, or user can define a customized formatter given a pattern.

category

Provides a list of pre-defined datetime formatter, SHORT, LONG, MEDIUM, etc.

Description

Allow user to choose from a list of pre-defined datetime formatter. Use a certain formatter to convert a "datetime" typed value using the given formatter.

pattern

Pattern of the formatter.

Description

Pattern of the date time formatter.
