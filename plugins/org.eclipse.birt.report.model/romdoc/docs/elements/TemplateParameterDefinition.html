TemplateElement

Represents a definition for a template element.

Description

Represents a template parameter definition element. A template parameter definition gives a definition of a template element. It can be referred by one template report item or one template data set. The template parameter definition has the following properties:

  • A required and unique name for this template parameter definition.
  • A type for this template parameter definition. It is a the enumeration of Table,Grid, Label, Text, ExtendedItem or other kind of report items and Dataset.
  • A static description message that will be show to the client user.
